 # Kafka Overview

## 1. Kafka의 탄생 배경
- 로그 수집 / 이벤트 스트리밍 필요성
- 기존 메시지 큐(ActiveMQ, RabbitMQ)와의 차이

## 2. Kafka 주요 특징
- 분산 로그 저장소
- 높은 throughput, 낮은 latency
- scale-out 구조

## 3. 핵심 개념 요약
- topic
- partition
- offset
- producer / consumer / broker

## 4. Kafka 주요 사용 사례
- 로그 수집
- 이벤트 기반 마이크로서비스
- 실시간 ETL / 스트리밍 파이프라인
